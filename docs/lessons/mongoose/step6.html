Install MongoDB Compass

Lesson

Learning Objectives

Students will be able to:

  • Run MongoDB Compass on their computer
  • Connect to a running instance via the MongoDB Compass GUI
  • Navigate the UI

Note

This video goes through using Compass with MongoDB installed locally on your computer. As noted in a previous lesson, we suggest using MongoDB Atlas, which is a cloud-hosted version of MongoDB. You can manage your collections and databases directly through the MongoDB Atlas web interface.
